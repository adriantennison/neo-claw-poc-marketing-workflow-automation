# Marketing Workflow Automation — AI Operations POC

POC showing how an AI + automation specialist could audit, score, and redesign marketing workflows for a high-volume client.

## Demonstrates
- Workflow audit scoring
- AI opportunity mapping
- Recommended automation blueprint
- Execution backlog output suitable for Zapier/n8n/CRM implementation

## Stack
- Node.js 20+
- Express

## Run
```bash
npm install
npm start
```

## Test
```bash
npm test
```
